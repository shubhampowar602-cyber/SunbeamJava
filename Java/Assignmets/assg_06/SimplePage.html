Introduction to web Devlopment

Web Devlopment is the process of creating websites and web applications and that run on the internet. It involves designing, coning, and maintaing web pages using various technologies.

Objectives of wev devlopmemnt

  • Understand basic structure of HTML document
  • Learn formating and text alignment
  • Use lists, images, and hyperlinks

Core Components of aweb Page

  1. HTML
    • Defines structure of the page
    • Uses tags like < html >, < head >, <body>
  2. CSS
    • Used for styling the web page
    • Controls layout and colors
  3. JavaScript
    • Adds interactivity
    • Handles events and validations

Text Formating Options

HTML allows text to be bold, italic, underline, and even colored to make the content attractive

Special Characters

Some special symbols used in HTML are:

  • Copyright symbol; ©
  • Registered trademark: ®
  • Trademark: ™
  • Less than sign: <
  • Greater that sign: >
  • Ampersand: &
